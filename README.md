# FPS

Mate todos os inimigos que aparecerem a sua frente. Cuidado para não perder toda sua vida ou acabar suas munições.

Controles:

Move - [A,W,S,D]

Move Camera - [Mouse]

Shoot - [Mouse Click]

Jump - [Space]

Run - [Shift]

Pause - [ESC] 

Referências: https://www.udemy.com/course/unity-first-person-shooter-tutorial/
